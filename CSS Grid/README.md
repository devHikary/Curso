# CSS Grid

* display: grid; 
* grid-template-areas: 
        "cabecalho"
        "conteudo"
        "rodape";
    grid-template-columns: auto;
    grid-template-rows: 50px 100vh auto;  
* .cabecalho {
    grid-area: cabecalho;
  }
*  grid-gap: .2rem; -->Adiciona um espaçamento entre linhas e colunas
* grid-column "start" / "end";
  grid-row "start" / "end";
* É possível haver um grid dentro de outro.
